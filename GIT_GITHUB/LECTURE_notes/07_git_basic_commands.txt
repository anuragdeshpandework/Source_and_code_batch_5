2 --------- RESTORING FILES -------------------

git checkout -f --> this will all restore your file to recent commit 


git checkout file path --> this will restore your given file to recent commit 

-----------------STATUS & HISTORY -------------------------

git status --> shows the current status of your working directory 


git log --> shows the commit history (detailed log)

git log -p  --> this shows the most recent commit with details 

git log --oneline --> shows you a compact history of the commit
